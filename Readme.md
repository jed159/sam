# one of wun photograghy website

## Brief Description of Application
A simple, visiually appearing photography website built using HTML and CSS to showcase most favorite photos and biography section together with contact form.
 
 Date 30 january 2026

## Description
 This application is a Photography project focused on showcasing and managing photographic content. It contains  displayed favorite photos collections, introduce short  biography of the creator and allow visitors to get in touch via contact form.

purpose of the project is to practice key web development skills including

-html basic structure
styling using CSS stylesheet
proper use of projects files and folder 
use of git and github
deploy a live site using github pages

##  Set up installation installation
Download repository from github
open the project folder using vs code
make sure the following exist: index.html-style.css-images folder containing images

## Features

-Display high-quality photographs
-Organize photos into categories or albums
-Simple and clean layouts
-Easy to customize and extend
